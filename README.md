# Adaptive-Sigmoid-Parametrized-Retrieval-Augmented-Generation-for-Cpld-Start-Recommendation

Read the full paper on IEEE Xplore: https://ieeexplore.ieee.org/document/11465948

ASR-RAG is an adaptive hybrid recommendation framework that integrates LightGCN collaborative filtering, Sentence-BERT semantic retrieval, and genre-aware constraints through a learnable sigmoid controller.
